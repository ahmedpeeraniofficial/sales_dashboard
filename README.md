# uber_dashboard

Built an interactive Power BI Sales Performance Dashboard to deliver clear and actionable financial insights for better business decision-making. The dashboard simplifies large and complex datasets into easy-to-understand visual reports.

It highlights key financial metrics including Gross Profit, Sales, Net Sales, and Cost of Goods Sold (COGS), enabling quick performance evaluation and financial analysis at a glance.

The dashboard is designed for fast, interactive analysis, helping users understand overall business performance, identify trends, and monitor financial health efficiently.

Why Power BI?

Converts raw data into meaningful insights
Interactive and user-friendly dashboards
Fast reporting and real-time analysis
Strong tool for business intelligence and financial tracking

This project demonstrates my ability to build data-driven dashboards that support strategic decision-making and financial analysis using Power BI.
